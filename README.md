## Projects Included

### 1) LockIn
A structured financial education platform with modular learning content and an interactive crisis decision simulator.  
**Stack:** Next.js, Supabase, Tailwind CSS

**Highlights**
- Modular learning flows for market fundamentals, personal finance, and investor psychology
- Crisis simulator that models long-term portfolio outcomes based on user decisions during historical market crashes

### 2) PaleoPath
An interactive human evolution explorer with a timeline, species pages, quizzes, and survival mode.  
**Stack:** React 18 + Vite, React Router v6, Framer Motion, Lucide React, localStorage

**Highlights**
- Progressive species experience: unique UI identity + quiz + survival mode per species
- Session-based progress: onboarding badge + localStorage scores (auto-reset for new user sessions)
